# regdat

Registry Hive Tool.

## dependencies
* offreg (part of WDK)

## supported features
* reg to dat
* dat to reg

## usage
```
Usage:
        regdat.exe --reg2dat --in_reg <path_to_reg_file> --out_dat <path_to_dat_file>
        regdat.exe --dat2reg --in_dat <path_to_dat_file> --out_reg <path_to_reg_file>
```
